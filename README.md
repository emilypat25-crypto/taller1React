# 📚 Taller 1 React — JavaScript Feature-Based

Proyecto de actividades JavaScript con estructura Feature-Based. Cada carpeta dentro de `src/features/` corresponde a una actividad independiente.

## 🗂️ Estructura

```
REACT_PROFE_FULL/
├── index.html                        ← Página de inicio
└── src/
    ├── shared/css/styles.css         ← Estilos globales
    └── features/
        ├── crud/                     ← Actividad 2
        ├── callback/                 ← Actividad 3
        ├── promise/                  ← Actividad 4
        ├── async/                    ← Actividad 5
        └── destructuring/            ← Actividad 6
```

## ✨ Actividades

| # | Tema | Descripción |
|---|------|-------------|
| 2 | CRUD | Insertar, editar, eliminar y exportar datos con `localStorage` |
| 3 | Callback | Búsqueda de países en dos bases de datos encadenadas |
| 4 | Promise | Verificar si una palabra termina en vocal con `resolve/reject` |
| 5 | Async/Await | Calcular el doble de un número con retraso de 2 segundos |
| 6 | Destructuring | Extraer propiedades de un objeto persona con ES6 |

## 🛠️ Tecnologías

- JavaScript ES6+
- HTML5 / CSS3
- Bootstrap 5
- localStorage

## 🚀 Cómo correr el proyecto

1. Clona el repositorio
```bash
git clone https://github.com/emilypat25-crypto/taller1React.git
```
2. Abre la carpeta en **VS Code**
3. Clic derecho en `index.html` → **Open with Live Server**
